# 项目定义：Fuli Plus

## 核心任务
构建一个面向地毯 / 织物图案设计的 agentic design system。系统不把用户的模糊表达直接翻译成一次性 prompt，而是先将用户的感受、偏好、选择与参考图映射为一组基于地毯设计语言的槽位状态（design state），再通过多轮推荐、反馈解释与状态更新，逐步收敛到更稳定、更可解释的设计方向。

## 要解决的核心问题
在传统图像生成或图片调节系统中，控制维度通常是固定且浅层的，例如饱和度、色调、滤镜、光影等。这类维度适合参数调节，但难以承接用户更模糊、更主观的审美表达，例如“更温暖一点”“不要太跳”“更有呼吸感”。

Fuli Plus 探索的是另一种机制：
- 不是让用户直接操作固定参数
- 而是把用户的模糊感受映射到一组更贴近地毯设计语言的槽位
- 用户每一轮的偏好选择都会反向影响这些槽位的状态
- 槽位状态进一步影响下一轮图片推荐、探索范围与整体收敛程度

## 核心差异
传统方案：固定控制项 -> 手动调节 -> 单次结果优化

Fuli Plus：模糊感受 -> 槽位映射 -> 多轮反馈 -> 状态更新 -> 逐步收敛

## 产品定位
这是一个把模糊审美意图转译为结构化设计状态，并通过反馈驱动收敛的创意设计智能体系统。
